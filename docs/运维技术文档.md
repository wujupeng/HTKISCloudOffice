## 1. 系统概述

HTKIS Cloud Office 是企业轻终端云办公平台，将安卓平板变成超轻终端，所有业务在云端运行。当前为 MVP 阶段，核心链路：

**安卓平板 Chrome → ****.htkis.com → Guacamole → RDP → Win Server → WPS → 文件编辑保存**

## 11. 故障排查

### 11.1 公网无法访问 ****.htkis.com

排查步骤:
1. 检查 DNS 解析: `nslookup ****.htkis.com` → 应为 x.x.x.214
2. 检查 HTTP: `curl -s -o /dev/null -w '%{http_code}' http://****.htkis.com/guacamole/` → 应返回 200
3. 检查 HTTPS: `curl -sk -o /dev/null -w '%{http_code}' https://****.htkis.com/guacamole/` → 应返回 200
4. 在 Ubuntu 本地检查 frps: `sudo systemctl status frps-htkis`
5. 在 Ubuntu 本地检查 nginx: `sudo nginx -t`
6. 检查 frpc: `ssh debian@192.168.x.102 "sudo systemctl status frpc"`
7. 检查 Guacamole: `ssh debian@192.168.x.102 "curl -s -o /dev/null -w '%{http_code}' http://127.0.0.1:8080/guacamole/"`

### 11.2 RDP 连接失败

排查步骤:
1. 检查 Win Server 是否可达: `ping 192.168.x.88`
2. 检查 RDP 端口: `nc -zv 192.168.x.88 3389`
3. 检查 guacd 日志: `docker logs guacd --tail 50`
4. 确认 security-mode=nla 在 Guacamole 连接配置中
5. 确认 Win Server 的 NLA 设置

### 11.3 frpc 无法连接 frps

```bash
# 检查 frpc 日志
sudo journalctl -u frpc -n 50

# 检查 x.x.x.214:7001 是否可达
nc -zv x.x.x.214 7001

# 重启 frpc
sudo systemctl restart frpc
```

## 14. 多实例部署指南

部署新客户实例只需修改 `config.env` 中的参数，每个客户使用不同的：

- **DOMAIN** — 域名（如 `cloudesk.customer-a.com`）
- **CUSTOMER_NAME** — 客户标识（如 `customer-a`）
- **FRP_REMOTE_PORT** — frp 远程端口（如 28080，避免冲突）
- **FRPS_PORT** — frps 端口（如 7002，避免冲突）
- **LAN_HOST** — 客户内网服务器 IP
- **RDP_HOST / RDP_USERNAME / RDP_PASSWORD** — 客户 Windows Server 信息

### 14.1 在线部署新实例

```bash
cd deploy
cp config.env.example config.env
vi config.env  # 填写新客户的参数
sudo bash install.sh
```

### 14.2 离线部署新实例

**打包（在有网络的机器上）：**
```bash
cd deploy
sudo bash build-offline.sh 1.0.0
# 生成 htkis-cloud-office-1.0.0-offline.tar.gz
```

**部署（在客户内网机器上）：**
```bash
tar xzf htkis-cloud-office-1.0.0-offline.tar.gz
cd htkis-cloud-office-1.0.0-offline/deploy
cp config.env.example config.env
vi config.env
sudo bash install-offline.sh
```

### 14.3 同一 Ubuntu 服务器多实例

如果多个客户的 frps 都在同一台 Ubuntu 上，需要避免端口冲突：

| 客户 | frps 端口 | frp 远程端口 | nginx 配置 |
|------|----------|-------------|-----------|
